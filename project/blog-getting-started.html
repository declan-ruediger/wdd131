Getting Started with Self-Hosting

What is self-hosting?

Self-hosting means instead of having your digital things saved on someone else's computer (i.e the "cloud"), you save them on your own computer. This has a few benefits:

  • Skips the middle-man of cloud providers and their payment fees
  • Keeps your stuff private (remember, if you're not paying for a cloud product, you are selling them your data and attention)
  • Allows for higher speeds and performance (you don't even need internet to access your stuff at home)

Where do I begin?

1. Select a computer

When your self-hosting, you'll use your own computer to host the data. This mean that computer has to be switched on for you to access the data on it. So, for example, if you use Immich to backup and organise your photos, you'll only be able to access those photos when the computer you're self-hosting them on is switched on.

Because of these, you ideally want to use a computer that's always on. We call this kind of computer a server.

2. Install Docker

First, you'll want to install Docker on your system. Docker is software that will manage the different self-hosted apps you want to use.

See Get Docker for how to get started. Docker Desktop is the most beginner-friendly way to use Docker.

3. Install Some Self-hosted Apps!

Now you're ready to install some apps! See the How-to and Blog for ideas.

Extra Tips
